# Visit Norrbotten

A tourism concept website built using the Bulma CSS framework.
The project focuses on responsive design, layout structure and reusable UI components.

![Preview](.preview.webp)

## Features

- Responsive layout using Bulma
- Reusable components via Bulma
- Multi-page structure
- Light/Dark mode toggle with JavaScript

## Tech

- HTML
- CSS (Bulma)
- JavaScript

## About the project

This project was built as part of a course assignment where the goal was to explore and use a CSS framework without modifying its core styles.
The focus was on understanding how to structure a website using predefined components and responsive breakpoints.

## What I worked on

- Build a responsive layout using Bulma
- Structuring content across multiple pages
- Implementing navigation and UI components
- Working within the constraints of a CSS framework
- Building a light/dark mode toggle via JavaScript
